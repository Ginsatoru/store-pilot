# StorePilot — Electron + React Dashboard

## Quick Start

```bash
npm install
npm run dev
```

## What it does
- `npm run dev` → starts Vite dev server + Electron window simultaneously
- `npm run build` → builds React then packages with electron-builder

## Stack
- Electron 28 · React 18 · Vite 5 · Tailwind CSS 3
